Elixir Counter - Setup & Docker Guide
-------------------------------------

Local Startup
-------------
python -m venv venv
venv\Scripts\Activate.ps1
pip install -r requirements.txt

python elixir_count.py

# Test with:
python test.py


Docker
------

# Build image:
docker build -t elixir_counter:latest .

# Run container:

# Publisher in same docker network
docker run --name elixir_counter --network cr_network -d elixir_counter:latest

# Use host networking (if network issues persist)
docker run --network=host --env-file .env -d elixir_counter:latest


Container Management
--------------------
docker stop elixir_counter
docker rm elixir_counter
docker rmi elixir_counter:latest
