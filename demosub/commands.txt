Card Detection - Setup & Docker Guide
-------------------------------------

Local Startup
-------------
python -m venv venv
venv\Scripts\Activate.ps1
pip install -r requirements.txt

python sample.py

# Test with:

python test.py


Docker
------

# Build image:
docker build -t <image>:latest .

# Run container:

# Publisher in same docker network
docker run --name <image> --network cr_network -d <image>:latest

# Use host networking (if netowrk issues persist)
docker run --network=host --env-file .env -d <image>:latest


Container Management
--------------------
docker stop <image>
docker rm <image>
docker rmi <image>:latest
