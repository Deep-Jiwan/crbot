Card Detection - Setup & Docker Guide
-------------------------------------

Local Startup
-------------
python -m venv venv
venv\Scripts\Activate.ps1
pip install -r requirements.txt

python card_detection.py

# Test with:

python card_receiver.py


Docker
------

# Build image:
docker build -t carddetection:latest .

# Run container:

# Publisher in same docker network
docker run --name carddetection --network cr_network -d carddetection:latest

# Use host networking (if netowrk issues persist)
docker run --network=host --env-file .env -d carddetection:latest


Container Management
--------------------
docker stop carddetection
docker rm carddetection
docker rmi carddetection:latest
